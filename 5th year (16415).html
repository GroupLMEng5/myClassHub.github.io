16415 - Computer Aided Engineering Design

×

Summary

CAED was a 20-credit class which ran over two semesters and was examined on three courseworks. Each coursework focussed on a different software: solidworks (CAD); ANSYS (FEA); and ANSYS Fluent (CFD). Each coursework was weighted equally to a total of 100% and required a design, analysis and a written report submission. The class is entirely based on software use and report-writing.

The students found refamiliarising themselves with the software straightforward but agreed that the courseworks were very time-consuming. For each section, some background lectures were held to reintroduce the software and cover the fundamental aspects of each analysis type. The class also included allocated lab times ran by tutors, however, the consensus was that there were not enough tutors to cater for the size of the class and working with peers was the best way to tackle the assignments.

Difficulty: 5.8
Overall Satisfaction: 5.9

- "Easy to grasp based on CAD/FEA skills"

- "Built upon skills previously gained in previous years. Coursework wasn't difficult; but the time it took was far too much for a 20 credit class. Think I ended up spending more time on that Creo coursework than on my dissertation in the end"

- "Not hugely difficult, but very time-consuming"

The difficulty level of the class was not high but the time required to be dedicated to it was. The course involves three separate courseworks which utilised various software to carry out an analysis to report on. The difficulty of the class was generated through the re-familiarisation with the different computer software.

- "Coursework covered various problems"

- "Good report centred assessment, with constant feedback."

- "There were 3 courseworks for the class, equally weighted. One was CAD, one was ANSYS FEA and the third was ANSYS CFD. Each one required design, analysis and then a report written about them. The reports format differed but were explained. More effort was required than expected. There was adequate feedback following each coursework mark given."

The entirety of the mark for this class came from the three courseworks. Each of these followed the same structure, consisting of a design aspect followed by an analysis to be reported on. This meant that the assessment was consistent and students knew what to expect. Varying difficulty and time dedicated to each coursework was experienced however and must be anticipated by students to ensure that they are not overloaded or caught off-guard by the workload.

- "Good more industrial feel thought it tied much of the mathematics and analysis into products."

- "Linked well in the end. Skill level was a step up from previous years and incorporated elements of different classes into the courseworks"

- "Important skills in CAD/FEA/CFD, linked in well with 4th and 5th year projects"

- "I do feel I learned a lot using the different softwares. Polished my computing skills."

- "This class required us to use skills previous learnt from Creo and Ansys but in saying that, we all needed to be reminded of these during the first few times using the software again - so don't worry if you've forgotten everything. There aren't any classes that use a lot (if any) Creo/CAD software or Ansys in 5th year, this is the last time I used them in uni. Some people used these skills for designing parts/testing their group projects. "

The class was very relevant to previous CAED classes and incorporated maths and analysis skills into a report based format. It was also good preparation for a dissertation or project work. Software and computer skills were developed which are essential to many roles in industry.

- "Good range of topics covered"

- "Lecture slides were intended to develop enough understanding of the topic to allow us to complete the assignment but they seemed unrelated a lot of the time. Projects tended to be just following step by step instructions as a result of this rather than requiring any creative thinking"

- "Past projects were more useful than official material"

- "Material was generally good but there was disparity between information provided on courseworks."

- "A lot of background material available"

Students thought that the class lectures focused more on developing background knowledge and competency on CAED concepts. The past projects made available provided the most useful information for tackling the courseworks.
